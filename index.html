Introduction

JavaScript is a programming language mainly used for web development.

It allows developers to build dynamic and interactive web pages.

JavaScript runs in all major browsers.

You can use it for frontend, backend, mobile apps, and more.

console.log("Hello, World!");
  • Easy to learn
  • Widely used
Variables

Variables store data values in JavaScript.

There are three main ways to declare variables: var, let, and const.

let is used for block-scoped variables.

const is used when you don't want the value to change.

let age = 22; const PI = 3.14;
  • var
  • let
  • const
Data Types

JavaScript has several basic data types.

The most common types include strings, numbers, booleans, and objects.

Arrays are special type of objects.

Null represents an empty value.

let name = "Noman"; let isAlive = true;
  • String
  • Number
  • Boolean
  • Object
Functions

Functions perform specific tasks.

You can reuse functions anywhere in your program.

A function can take parameters and return values.

Arrow functions are a modern syntax.

function greet() {
  return "Hello";
} const add = (a, b) => a + b;
  • Reusable code
  • Easy to maintain
  • Arrow functions
Loops

Loops repeat actions multiple times.

The most common loops are for and while loops.

Loops help avoid repeating code manually.

Array loops are very popular.

for (let i = 0; i < 5; i++) {
  console.log(i);
} let i = 0;
while (i < 5) { i++; }
  • for loop
  • while loop
  • forEach loop
